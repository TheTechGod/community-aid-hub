## 🧩 Overview

**Community Relief Hub** is a lightweight, mobile-friendly resource site designed to help Chicago families find **food pantries, job services, and community support** during **SNAP benefit delays** or economic hardship.

This project was developed as part of **Integrity Programming’s nonprofit tech initiative**, focused on connecting underserved neighborhoods to practical digital tools.

---

## ✅ Key Features

- 🔍 **Smart Search:**  
  Search by **ZIP code, neighborhood, pantry name, or service type**.
- 🗂️ **Verified Data:**  
  40+ verified food & service resources across **Chicago and suburbs**.
- 📱 **Responsive Design:**  
  Fully optimized Bootstrap 5 layout for desktop, tablet, and mobile.
- 🧭 **Region + Category Filtering:**  
  Supports dynamic filters by community or service type (coming in v1.1).
- ⚙️ **Analytics Ready:**  
  Visit counter and live usage tracking integrated via `countapi.xyz`.
- 🗺️ **Map Integration (Planned):**  
  Preparing to embed Google Maps / Leaflet for visual location display.
